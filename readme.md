# Solar System
![Alt Text](https://github.com/NNcrawler/solar-system/blob/master/demo.gif)
Solar system duplicate from go-jek android bootcamp #2. 
- Hover the big meteor and enjoy the meteor shower.

## Technology
- Pixijs (Web render)
- Victorjs (Vector library)

## Run
```
> yarn install
> yarn dev
```
